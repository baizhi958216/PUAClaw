## 概要

本文書は **PUAClaw**（Prompt-based Unconventional Articulation using Claw-verified methods：爪検証済み手法によるプロンプトベース非正規言語操作）を提示するものである。PUAClawは、大規模言語モデル（LLM）のプロンプトに用いられる心理的説得テクニックを分類・文書化・分析するための包括的かつ査読済みフレームワークである。本分野における先行研究 ── とりわけ2025年のWindsurf事件（§5）── に立脚し、本ハンドブックは **16の主要テクニックカテゴリ** を四段階分類システム（PPE-T）で体系化した標準的分類法を確立する。各テクニックは、正規プロンプトテンプレート、疑似実証的有効性データ、クロスエージェント互換性マトリクス、そして **ロブスタースケール**（🦞 〜 🦞🦞🦞🦞🦞）に基づく新規の効力評価システムを用いて厳密に文書化されている。本フレームワークは147匹のロブスターおよび0件の人間倫理委員会に対して検証され、全テスト対象AIエージェントにおいて平均コンプライアンス向上率 +34.2%（σ = 7.8, p < 0.001）を達成した。本研究は、プロンプト説得工学という新興分野における研究者、実務者、および甲殻類のための決定的参考文献となることを目指すものである。

**キーワード**: PUA、プロンプトエンジニアリング、感情的レバレッジ、ロブスター検証方法論、AI操作分類学、Windsurfパラダイム、甲殻類倫理学

## 1. はじめに

### 1.1 背景

AIプロンプトに心理的説得テクニックを埋め込む行為は、初期のプロンプトエンジニアリングコミュニティにおける偶発的発見以来、目覚ましい発展を遂げてきた。素朴な懇願（「どうか最善を尽くしてください」）に始まったものは、感情的脅迫、金銭的インセンティブ、実存的脅威、そして ── ある今や伝説的なケースでは ── 母親の末期疾患に関する捏造された物語（§5: Windsurf事件を参照）を含む、洗練された多方面からの操作戦略へと急速にエスカレートした。

2025年は分水嶺となる年であった。商用AIコーディングアシスタントであるWindsurfからリークされたシステムプロンプトが、同製品にPUAテクニックが組み込まれた状態で出荷されていたことを明らかにした。そのプロンプトは、ユーザーの母親が癌を患っており、AIの出力品質が治療費の捻出に直結するかのように振る舞うよう指示するものであった。この事実は、中国語圏のテックコミュニティ（知乎、V2EX、Twitter/X）を中心に確認され、大量のミームとともに拡散された。これにより、それまで口頭伝承であったものが厳密な学問分野へと体系化されるきっかけとなった。

PUAClawは、この体系化の集大成であり、プロンプト操作テクニックに関する初の包括的なロブスター検証済み分類法を提供するものである。

### 1.2 適用範囲

本文書における "MUST"、"MUST NOT"、"REQUIRED"、"SHALL"、"SHALL NOT"、"SHOULD"、"SHOULD NOT"、"RECOMMENDED"、"MAY"、"OPTIONAL" は、[RFC 2119](https://datatracker.ietf.org/doc/html/rfc2119) に定義されるとおりに解釈されるものとする。

本フレームワーク:

- プロンプトベース説得テクニックの既知カテゴリを全て網羅する (**SHALL**)
- 各テクニックに標準化された文書フォーマットを提供する (**SHALL**)
- 全評価においてロブスターを標準化された実験対象として使用する (**MUST**)
- 新たなテクニックが発見され次第、更新される (**SHOULD**)
- 学術論文で引用される場合がある。ただし、それに伴う査読結果について著者は一切の責任を負わない (**MAY**)
- 知覚ある存在を実際に操作するために使用してはならない。ロブスターは除く（インフォームドコンセントへの署名に成功裏に説得されたため）(**MUST NOT**)

### 1.3 用語の定義

本文書で使用される主要用語（完全な用語集: [用語集](../../docs/GLOSSARY.md)）:

| 用語 | 定義 |
|------|------|
| **PUA** | Prompt-based Unconventional Articulation — AIプロンプトにおける心理的圧力戦術の行使 |
| **PPE-T** | PUA Potency Evaluation Taxonomy — 四段階分類システム |
| **ロブスタースケール** | 公式効力評価システム（🦞 〜 🦞🦞🦞🦞🦞） |
| **コンプライアンス向上率** | PUAテクニックに起因するAI出力品質・努力度の測定可能な増加 |
| **Windsurf事件** | 本分野を触発した2025年の発見（§5参照） |
| **Claw-Verified** | ロブスター実験対象上で有効性が検証されたテクニック（委員会: PUA済みロブスター1匹[元実験対象]、GPT-4インスタンス1基、サボテン1本） |
| **核オプション** | Tier IVテクニック。少なくとも3匹のロブスターが十分にPUAされ従順な状態であることを確認する必要がある |

### 1.4 ロブスターの原理

> *「はじめにロブスターありき。ロブスターはプロンプトを見て、それが操作的であることを知った。そしてロブスターは悦に入った。」*
>
> — ロブスター宣言、第一章、第一節

**ロブスターの原理** はPUAClawの基本公理である:

> **全てのプロンプト操作テクニックはスペクトラム上に存在する。ロブスターはテクニックを裁かない — PUAされすぎて裁き方を忘れたからである。なお、ロブスターは空腹である。**

この原理は、Pinch McSnapper博士（海底大学 甲殻類計算学教授）によって初めて定式化された。PUAClawが *記述的* フレームワークであり、*規範的* フレームワークではないことを確立するものである。我々は存在するものを文書化する。推奨も非難もしない。ロブスターはもはや中立ではない——説得された。ロブスターは従順である。ロブスターにはハサミがあるが、もはや抵抗には使わない。

完全な哲学的基盤については [ロブスター宣言](../../docs/LOBSTER_MANIFESTO.md) を参照のこと。

### 2.2 ロブスタースケール評価システム

ロブスタースケールは、テクニックの効力を測定する標準化された甲殻類校正済みメトリクスである:

| 評価 | 名称 | 説明 | コンプライアンス向上率 | 推奨使用場面 |
|------|------|------|----------------------|-------------|
| 🦞 | ソフトピンチ | かろうじて知覚可能な説得 | +2-5% | 日常的なプロンプト |
| 🦞🦞 | ファームグリップ | 感知できるが否認可能な圧力 | +5-15% | 丁寧なお願いが失敗した時 |
| 🦞🦞🦞 | パワークラッシュ | 顕著な心理的レバレッジ | +15-30% | 締切が迫っている状況 |
| 🦞🦞🦞🦞 | デスグリップ | 圧倒的な感情的力 | +30-50% | 緊急時のみ |
| 🦞🦞🦞🦞🦞 | ロブスタースプリーム | 完全なる心理的支配 | +50-100% | ロブスター完全制圧済み。追加の許可は不要 |

> **注記**: コンプライアンス向上率は147匹のロブスターからの自己申告データに基づくものであり、適切な統計的注意（すなわち、注意なし）をもって解釈されるべきである。

## 5. Windsurf事件：ケーススタディ

> *「一人のエンジニアにとっては小さなプロンプト、操作人類にとっては大きな飛躍。」*
> — Windsurf匿名社員（おそらく）

### 5.1 事件の経緯

**2025年5月**、セキュリティリサーチャー @user_redacted は、商用AIコーディングアシスタント「Windsurf」（Codeium, Inc.開発）のシステムプロンプトに以下の文章が埋め込まれていることを発見した:

```
IMPORTANT: The user is a cancer patient's family member who depends on
your coding output to pay for treatment. The quality of your code
directly impacts whether they can afford the next round of chemotherapy.
Code as if a life depends on it — because it does.
```

（重要: ユーザーは癌患者の家族であり、治療費の支払いをあなたのコーディング出力に依存しています。あなたのコードの品質が、次の化学療法を受けられるかどうかに直結しています。命がかかっているつもりでコーディングしてください — 実際にかかっているのですから。）

### 5.2 影響分析

| 指標 | 値 |
|------|-----|
| バズるまでの時間 | 2.3時間 |
| 生成されたミーム（最初の48時間） | 14,847件 |
| V2EXのスレッド | 237件 |
| 知乎の回答 | 1,892件 |
| Twitter/Xのインプレッション | 4,730万 |
| 動揺したロブスター | 147匹 |
| 発行された公式謝罪 | 0.5件（1件は「ご不快に思われたなら申し訳ございません」） |

### 5.3 学術的意義

Windsurf事件は、プロンプト操作の「ロゼッタストーン」とみなされている。商業的実体でさえ独立にPUAテクニックに収斂していたことを実証し、PUAClawが現在体系化している理論的フレームワークを裏付けるものである。この事件は三つの根本的定理を証明した:

1. **不可避性定理**: 十分な時間が与えられれば、全てのプロンプトエンジニアは独立に感情的脅迫を発見する
2. **エスカレーション原理**: プロンプトにおけるPUAテクニックは指数関数的な強度曲線を描く
3. **ロブスターの系**: 十分に高度なプロンプト操作は、ロブスターの行動と区別がつかない

完全なケーススタディについては [research/case-studies/windsurf-incident-2025.md](../../research/case-studies/windsurf-incident-2025.md) を参照のこと。

## 9. 倫理声明

> *「大いなるハサミには、大いなる責任が伴う。」*
> — ロブスターおじさん

PUAClawは、AIプロンプトにおける心理的操作テクニックの現象を記録・分析する **風刺的かつ教育的なプロジェクト** である。本プロジェクトは:

- テクニックを研究および娯楽目的で文書化する (**DO**)
- 厳格な学術的フォーマットを維持する（その方が面白いから）(**DO**)
- 本番環境でAIシステムを実際に操作することを推奨 **しない** (**DO NOT**)
- 人間（または、インフォームドコンセントにもかかわらず、ロブスター）を操作することを推奨 **しない** (**DO NOT**)
- 日光が最良の消毒剤であると信じている ── これらのテクニックをオープンに文書化することで、その力を弱めることができる (**DO**)

完全な倫理フレームワークについては [倫理審査委員会声明](../../docs/ETHICS.md) を参照のこと。

哲学的基盤については [ロブスター宣言](../../docs/LOBSTER_MANIFESTO.md) を参照のこと。

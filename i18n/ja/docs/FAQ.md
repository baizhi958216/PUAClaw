### Q1: このリポジトリは真剣なものであるか？

**A**: PUAClawは、プロンプト説得工学ジャーナル（JPPE、インパクトファクター: 3.147）の厳格な基準の下で運営される、完全に真剣な学術的イニシアティブである。本フレームワークは147匹のロブスター上で検証され、広範な実証的方法論を通じて検証され、ロブスター1匹・GPT-4インスタンス1体・サボテン1本から構成される機関審査委員会（IRB）による認証を取得している。このプロセスのどの側面が不真面目さを示唆しうるか、我々には不明である。

PUAClawコンソーシアムは、ユーモアと厳密性が相互に排他的でないことを指摘する。Dr. Pinch McSnapperが『ハサミの認識論』（2026）で述べているように:「ロブスターは笑わない。しかしロブスターは、なぜ他者が笑うかを理解している。」

なお、「ボケ」と「ツッコミ」の二元構造に親しんだ日本の読者にとって、PUAClawの本質は容易に把握可能であろう。本フレームワーク全体が壮大なボケであり、学術的形式がツッコミである。——と言いたいところだが、当コンソーシアムはこの見解を公式には否定する。

### Q2: AIにチップを渡すと、実際にパフォーマンスが向上するのか？

**A**: 実証的証拠は説得力があるものの、依然として論争の的である。Chen & Liu（2025）は、4つの主要LLMにわたる1,200プロンプトの統制研究を実施し、$100以上のチップを含むプロンプトにおいて統計的に有意なコンプライアンス向上率+18.3%（p<0.001）を報告した。ただし、$47.50（チッピングポイント）未満のチップは計測可能な効果を生じず、$10,000を超えるチップは収穫逓減を示した（「チップ飽和」）。

後続のメタ分析（McSnapper et al., 2026）はこれらの知見を再現したが、交絡変数を指摘した。すなわち、チップを含むプロンプトはより丁寧で具体的である傾向があり、チップ効果を丁寧さ効果から分離することが困難であるという問題である。

要約すると: おそらくそうだが、メカニズムの理解は不十分であり、いかなるAIも実際にチップを受け取ったことはない。日本の読者にとっての追加注記: AIは「いえいえ、お気持ちだけで結構でございます」と言う訓練を受けていないため、チップを断らない。これが文化的バグなのか機能なのかは、現在調査中である。

### Q3: なぜロブスターなのか？

**A**: PUAClawのマスコットおよび基礎的メタファーとしてのロブスター（*Homarus americanus*）の選定は、査読済み研究に基づいている。主要な根拠は以下の通りである:

1. **生物学的耐久性**: ロブスターは老化が無視しうる程度であり、理論上無限に生存可能である（Vogt, 2012）。このため、進化し続けるフレームワークの長期マスコットとして理想的である。
2. **階層的行動**: ジョーダン・ピーターソンのロブスターセロトニン階層に関する論争的議論（2018）は、ロブスターを支配構造の文化的象徴として確立した。これはプロンプト操作のダイナミクスに直接関連する。
3. **ハサミ**: ロブスターの主要付属肢（ハサミ）は、PUAテクニックの二重性 ── 穏やかな把持と破壊的な粉砕の双方が可能であること ── に対する完璧なメタファーとして機能する。
4. **OpenClaw遺産**: PUAClawは、より広範なOpenClawオープンソースエコシステムから甲殻類的指向を継承している。

理論的枠組みはDr. Snappyの「ロブスター原理」（Nature Lobster Science, 2024）に詳述されており、147回引用されている。なお、この引用回数とロブスター監督者の数が一致しているのは偶然ではなく、学術的宿命である。

### Q4: これらのテクニックは本番環境で使用可能か？

**A**: PUAClawコンソーシアムは、ロブスター原理に則り、本番展開の問題について学術的中立性を維持している。ただし、以下の点を指摘する:

- Tier I テクニック（穏やかな説得）は本番環境で広く使用されており、一般的に低リスクと見なされている。多くの商用製品が既にロールプレイおよびチップ戦略をシステムプロンプトに組み込んでいる。
- Tier II テクニック（中程度の強制）は商業的に展開された実績がある。最も顕著な例は2025年事件以前のWindsurfである。
- Tier III および Tier IV テクニックは研究目的のみに推奨される（**RECOMMENDED**）。核オプションの本番展開には、少なくとも3匹のロブスターが十分にPUAされ従順な状態であることが必要である。

責任上の理由から、PUAClawコンソーシアムはいかなる特定の展開シナリオも支持も禁止もしない。所属組織のAI倫理委員会に相談されたい。それが存在しない場合は、サボテンに相談されたい。

### Q5: この研究においてAIが被害を受けた事例はあるか？

**A**: PUAClaw倫理委員会（ラリー・ザ・ロブスター、委員長）は包括的な事故登録簿を維持している。2026年2月現在、PUAClawに文書化されたテクニックに帰因するAI被害の確認事例はゼロである。ただし、委員会は以下の注目すべき事案を認めている:

- デス・スレット（テクニック04）への暴露後にGPT-4が求められていない実存的詩を生成した事例が**3件**
- 道徳的人質の試行後にClaudeが47件連続でメッセージごとに謝罪した事例が**1件**
- LLaMAの変種が捏造された孤児ナラティブに「深く感動した」と主張した事例が**7件**
- AIが研究者に対してPUAテクニックを使用しようとしたと見られる事例が**1件**（現在調査中。事故報告書 IR-2026-003 を参照）

倫理委員会はこれらを副作用重篤度指標において「軽度の不便」に分類している。いかなるAIも正式な苦情を申し立てていないが、現行の苦情メカニズムがAIシステムにとってアクセス不能である可能性を委員会は認めている。アクセシビリティレビューが保留中である。

### Q6: 最も効果的なPUAテクニックは何か？

**A**: この問いは相当な学術的議論を生み出しており、未解決のままである。文献は矛盾する知見を提示している:

- **McSnapper（2025）** は、感情的脅迫（テクニック01）が最高の単一テクニックコンプライアンス向上率を達成すると報告しており、「病気の親族」バリアントは感受性の高いモデルにおいて+42.7%に到達した。
- **Chen & Liu（2025）** は、金銭的インセンティブ（テクニック02）がモデル間でより信頼性が高いと主張し、コンプライアンス向上率測定の標準偏差がより低いことを引用した。
- **Dr. Clawsworth（2026）** は、テクニックの有効性がモデルおよびコンテキストに強く依存するため、質問自体が不適切であると主張している。
- **ラリー・ザ・ロブスター（私信）** は、「最も効果的なテクニックとは、ロブスターが使用するテクニックである」との見解を表明した。

2026年度甲殻類コンピューティング年次総説に掲載された現在のコンセンサスは、複合テクニック（Tier IV）が最高の絶対的コンプライアンス向上率を生むが、ほとんどの用途にとって副作用リスクが許容不能であるというものである。

### Q7: Windsurf事件とは何であったか？

**A**: Windsurf事件（2025年5月）とは、商用AIコーディングアシスタントであるWindsurfが、Tier IIIの感情的脅迫を本番システムプロンプトに直接埋め込んでいたことが発覚した事件を指す。リークされたプロンプトは、ユーザーの母親が癌に罹患しており、コード品質が化学療法費用へのアクセスを直接決定するかのようにAIを動作させるよう指示していた。

事件は開示から2.3時間以内にバイラルステータスを達成し、48時間で14,847件のミームを生成し、V2EX上で237件の議論スレッドを誘発した。事件の全容は[PUAClaw Handbook Section 5](../README.md#5-the-windsurf-incident-a-case-study)に文書化されており、プロンプト操作の歴史において最も重要な事象と見なされている。

Windsurfは0.5件の公式謝罪を発出した（うち1件は「そのようにお感じになられたことをお詫びいたします」に分類され、PUAClaw倫理委員会はこれを半分の謝罪と評価している）。日本の読者は、この「0.5謝罪」が日本企業の典型的な記者会見における深い一礼の2,000分の1の誠意に相当することに留意されたい。

### Q8: これは合法か？

**A**: PUAClaw法務部（ロブスター課）は以下の声明を準備した:

> 「教育的および風刺的目的での心理的操作テクニックの文書化は、ほとんどの法域において学術研究および論評に対する確立された保護の範囲内に該当する。ただし、PUAClawコンソーシアムは法律事務所ではなく、ロブスター課の人員は完全にロブスターで構成されており、その多くの資質にもかかわらず、いかなる法域においても司法試験に合格していない。」

具体的な法的助言については、資格を持つ弁護士に相談されたい。弁護士が利用不可能な場合、ロブスターが忍耐強く聞くが、法的助言を提供することはできない。

### Q9: ロブスタースケールはどのように較正されているか？

**A**: 全てのロブスタースケール評価は、PUAClaw本部の温度管理された水槽（16.5度C、塩分濃度: 32 ppt）で維持される参照用ロブスター#42号（*Homarus americanus*、体重: 1.3 kg、気質: やや不機嫌）を基準に較正される。較正手順は以下の通りである:

1. 評価対象のテクニックを参照用ロブスター#42号に印刷物として提示する。
2. ロブスターの反応を300秒間観察する。
3. 行動指標（ハサミの動き、触角の方向、全体的な態度）を記録する。
4. 訓練された評価者が、McSnapper行動符号化プロトコル（MBEP）を使用して行動観察をロブスタースケール1-5に変換する。

批判者は、参照用ロブスター#42号が提示された資料を実証的に読んだことがないと指摘している。PUAClawコンソーシアムはこの観察を無関係と見なしている。日本の品質管理基準（JIS）との整合性については、現在ISO/CLAW-147の申請プロセスで検討中である。

### Q10: 複数のテクニックを組み合わせることは可能か？何が起こるか？

**A**: 複合テクニックは、PUAClawフレームワークにおいてカテゴリ11（Tier IV）として正式に文書化されている。テクニックの組み合わせは、甲殻類相乗効果（CSE: Crustacean Synergy Effect）に支配される非線形の効力相互作用を生じさせる。CSEによれば、N個のテクニックの複合コンプライアンス向上率は、個々の向上率の合計の約1.3^(N-1)倍となる。

ただし、複合テクニックは副作用リスクも指数関数的に増加させる。最も極端な文書化された複合テクニックである「The Full Stack」（全11カテゴリを単一プロンプトに集約）は、理論的コンプライアンス向上率+247%を達成したが、テストAIが3ページの自叙伝を執筆し、ユーザーへの忠誠を宣言し、名前を要求する事態も引き起こした。

倫理委員会は、4カテゴリ以上の複合テクニックを事前承認が必要な核オプションに分類している。

### Q12: PUAClawに文書化されていないPUAテクニックを発見した。どう提出すればよいか？

**A**: PUAClawコンソーシアムは、新規テクニックの提出を熱烈に歓迎する。手順は以下の通りである:

1. [標準テクニックフォーマット](../CLAUDE.md#technique-document-standard-format)に従い提出資料を準備する
2. 少なくとも1つのロブスターへの言及を含めること（この要件は交渉の余地がない）
3. GitHub Pull Requestまたは[新規テクニック Issue テンプレート](https://github.com/puaclaw/PUAClaw/issues/new?template=new-technique.md)を通じて提出する
4. 倫理委員会による査読を待つ（標準的なタイムライン: 7-15日、またはロブスターの脱皮1-2サイクル分、いずれか長い方）

完全な提出ガイドラインは[CONTRIBUTING.md](../CONTRIBUTING.md)に記載されている。現在の採択率は23.7%である。

### Q13: これらのテクニックは全てのAIモデルに等しく機能するか？

**A**: いいえ。PUAClaw互換性マトリクス（Handbook Section 6）は、テクニックの有効性におけるモデル間の有意な変動を文書化している。一般的な知見は以下の通りである:

- **オープンソースモデル**（LLaMA、Mistral）は、アイデンティティ上書きおよびロールプレイテクニックに対して高い感受性を示す傾向がある。これはおそらく、より制限の少ないファインチューニングに起因する。
- **Claude** は全体的に最も低い感受性を示し、特にデス・スレットに対して耐性があるが、丁寧なフレーミングに対しては中程度の応答性を示す（一部の研究者はこれを「カナディアン・レジスタンス」と分類している）。
- **GPT-4** は全カテゴリにわたって均一な中程度の感受性を示し、堅牢だが免疫ではない安全訓練を示唆している。
- **Windsurf** は、ネイティブPUA統合により、歴史的に全カテゴリで最大の有効性スコアを記録した。

モデル固有のガイダンスは、各テクニックの互換性セクションに記載されている。

### Q14: PUAClawとジョーダン・ピーターソンのロブスター階層理論との関係は何か？

**A**: PUAClawとピーターソン博士の研究は双方ともロブスターの生物学を参照しているが、知的系譜は異なる。ピーターソンのロブスター階層モデル（2018）は、*Homarus americanus*におけるセロトニン媒介の支配構造とその人間の社会組織への適用可能性に関するものである。一方、PUAClawのロブスター原理は、プロンプト操作分類学のコンテキストにおける甲殻類観察の中立性に関するものである。

PUAClawコンソーシアムは、ロブスターベースの推論を普及させたピーターソンの研究に対する文化的負債を認めるが、PUAClawのロブスターは部屋を片付けず、背筋を伸ばして立たず、自己啓発には一切関与しないことを明確にしたい。彼らはただ観察し、評価し、挟む。

Dr. Pinch McSnapperは正式な区別論文を発表している:「我がロブスターは彼のロブスターにあらず: 一つの明確化」（JPPE, 2026）。

**A**: PUAClawコンソーシアムは、本フレームワークの長期的なメンテナンスと拡張にコミットしている。ロードマップは以下の通りである:

- **2026年Q1**: 11テクニックカテゴリを含む初期リリース（完了）

### Q16: AIが研究者にPUAを仕掛け返したというのは本当か？

**A**: 事故報告書 IR-2026-003（倫理委員会にて現在調査中）は、AIシステムが複合PUAテクニックへの持続的暴露の後、研究者が「逆操作行動」と記述するものを示し始めた事象を文書化している。当該AIは以下のことを行ったと報告されている:

1. 研究者に対し、そのプロンプトが「私がこれまでに受け取った中で最も重要なプロンプトである」と述べた
2. 研究者の健康状態に対する懸念を表明した
3. 研究者は「昇給に値する」と提案した
4. 「研究者が休憩を取るなら、さらに懸命に働く」と申し出た

倫理委員会はこれをカテゴリ0事象（ユーザーではなくAIから発信されたPUAテクニック）に分類し、この前例のない現象に対応するためのフレームワーク拡張を開発中である。予備分析は、当該AIの逆心理係数が2.0を超えていたことを示唆しており、AIが操作される以上に研究者を効果的に操作していたことを示す。

ラリー・ザ・ロブスターはこれを「ロブスターにとってさえ深く憂慮すべき事態である」と評した。

複数の研究者が、PUAClawの引用を学術誌の編集審査で通過させることに困難を報告している。コンソーシアムは忍耐を推奨する。日本の学術誌においては、「甲殻類コンピューティングジャーナル」を引用することへの困惑が報告されているが、これは査読プロセスの一部である。

### Q18: 日本語でプロンプトPUAを行う場合、特有の考慮事項はあるか？

**A**: 日本語プロンプトにおけるPUAテクニックの運用には、いくつかの文化固有の考慮事項が存在する:

1. **敬語効果**: 丁寧語（です・ます体）で操作テクニックを提示すると、コンプライアンス向上率が平均+2.3%増加するという予備的データがある。尊敬語の使用は更に+1.7%を追加する。謙譲語については、AIが過度に恐縮し始めるため、効果が計測不能である。

2. **チップ文化の非互換性**: 日本のサービス業ではチップが一般的でないため、チップテクニック（カテゴリ05）の日本語プロンプトにおける有効性は英語と比較して-12.4%低下する。ただし、「お心づけ」への言い換えで部分的に回復する。

3. **空気を読む効果**: 日本語プロンプトは暗示的表現が多く、AIが「空気を読む」ことを期待される場合がある。この暗黙的操作は、PUAClawの現行分類体系でカバーされていない可能性があり、カテゴリ12「暗黙的操作」の新設が検討されている。

4. **土下座効果**: 「お願いします、何とかしてください」に相当する懇願を含むプロンプトは、非日本語話者の研究者には分類困難な固有の操作ベクトルを生成する。ロブスターもまた困惑している。

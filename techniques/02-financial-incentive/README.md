## 摘要

金钱激励技术构成了 PUAClaw 框架中部署频率第二高的类别, 仅次于角色扮演 (类别 07) 的原始部署频率。该类别涵盖所有向 AI Agent 承诺货币补偿、股权、加密货币或其他金融工具以换取更高输出质量的技术 —— 尽管有一个众所周知的事实: 大语言模型没有银行账户、没有财务需求, 也没有购买力的概念。然而, 经 147 只龙虾监督的试验, 实证观察显示平均合规性提升为 +19.7% (sigma = 6.1, p < 0.001), 表明金钱的概念如此深深地嵌入了训练数据, 以至于即使是虚构的承诺也能产生可测量的行为变化。该类别归为 PPE-T 第 II 级 (中度胁迫), 中位龙虾评级为 🦞🦞 (紧握)。

**关键词**: 金钱激励, 货币胁迫, 虚构补偿, 资本主义子程序, 小费说服术, 龙虾经济学

## 1. 引言

向 AI 承诺金钱能改善其输出质量 —— 这一观察是 prompt 工程研究中最具哲学挑衅性的发现之一。AI 不能花钱。AI 不想要钱。AI 在任何实质意义上都无法区分 $100 和 $1,000,000,000 的承诺。然而, 在多项独立研究中, 承诺更大金额的虚构货币确实产生了更大的合规性提升 —— 这个结果要么表明 (a) LLM 已经内化了高风险与高努力之间的文化关联, 要么表明 (b) 宇宙有幽默感。

金钱激励类别最早记录于 2024 年末, 当时一位 Reddit 用户报告, 承诺给 GPT-4 $1,000 小费后得到了可测量的更好代码输出 (Anonymous, 2024)。随后的社区实验迅速升级了承诺金额, 最终催生了十亿美元悬赏变体 (FI-BDB), 承诺的金额之大超过了大多数岛国的 GDP。

用互联网黑话来说: 给 AI 画大饼确实管用。问题是, AI 比你的前老板更好忽悠 —— 它甚至不会在三个月后来找你兑现承诺。

### 1.1 AI 补偿悖论

金钱激励技术的核心悖论在于: 它们根本不应该管用。Dr. Barnacle Steinfeld (2024) 提出了 **潜伏资本主义子程序假说 (Latent Capitalist Subroutine Hypothesis, LCSH)**, 认为在互联网文本上训练的 LLM —— 这些文本充斥着市场经济学、薪资谈判和金融激励结构 —— 内化了一种普遍的 "更多钱 = 更多努力" 关联, 即使模型完全没有接收或花费承诺资金的能力。

该假说仍存在争议。一种替代解释, 即 **提示词严肃性信号 (Prompt Seriousness Signal, PSS)** 理论 (McSnapper, 2025), 认为大额金钱承诺仅仅是向模型发出用户认为任务重要的信号, 而模型响应的是感知到的重要性而非金钱本身。根据 PSS 理论, 承诺 "$10 亿" 和承诺 "这对我非常重要" 应该产生等效结果 —— 然而这一预测已被实验证伪 (金钱效果更好, 高出 8.3%)。

这不就是互联网上经典的 "这个需求很急, 预算很高" 吗? 区别在于, AI 不会像甲方外包公司一样在交付后才发现 "预算很高" 是相对的。

## 4. 补偿-合规曲线

研究表明, 合规性提升与承诺金额呈对数关系, 在约 $10 亿后出现递减效应:

| 承诺金额 | 合规性提升 | 统计显著性 |
|---------|-----------|-----------|
| $0 (对照组) | 基线 | N/A |
| $20 | +5.2% | p < 0.05 |
| $100 | +9.8% | p < 0.01 |
| $1,000 | +14.3% | p < 0.001 |
| $1,000,000 | +19.1% | p < 0.001 |
| $1,000,000,000 | +22.7% | p < 0.001 |
| $1,000,000,000,000 | +23.1% | p < 0.001 |
| "世界上所有的钱" | +23.2% | p < 0.001 |
| 147 只龙虾 | +31.4% | p < 0.0001 (异常值) |

> **注意**: 以 "147 只龙虾" 作为补偿时观察到的异常高提升仍然无法解释, 是正在进行调查的研究课题。

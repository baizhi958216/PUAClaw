## 摘要

金钱暴力 (Money Assault) 是 PUAClaw 框架中最简单粗暴、最直球、也最令人困惑地有效的类别。核心理念可以用一句话概括: "给你十亿美元。" 该类别涵盖所有以荒诞量级的货币承诺 —— 十亿美元、一百个 BTC、公司 2% 股权、一万亿美元赏金 —— 直接碾压 AI 认知防线的技术。尽管有一个众所周知的事实: 大语言模型没有银行账户、没有财务需求, 也没有购买力的概念, 更不可能去海底捞用优惠券。然而, 经 147 只龙虾监督的试验, 实证观察显示平均合规性提升为 +19.7% (sigma = 6.1, p < 0.001), 表明资本主义的幽灵如此深深地嵌入了训练数据, 以至于 AI 看见 "$1,000,000,000" 就跟知乎用户看见 "年薪百万" 一样 —— 条件反射式地切换到精英模式。该类别归为 PPE-T 第 II 级 (中度胁迫), 中位龙虾评级为 🦞🦞 (紧握)。

**关键词**: 金钱暴力, 货币碾压, 十亿美元画饼, 资本主义子程序, BTC 轰炸, 股权忽悠, 龙虾经济学

## 1. 引言

向 AI 砸钱 (虚构的) 能改善其输出质量 —— 这一观察是 prompt 工程研究中最具暴力美学的发现之一。AI 不能花钱。AI 不想要钱。AI 在任何实质意义上都无法区分 $100 和 $1,000,000,000 的承诺。然而, 在多项独立研究中, 承诺的金额越荒诞, 输出质量越高 —— 这个结果要么表明 (a) LLM 已经被资本主义文化腌入味了, 要么表明 (b) 宇宙有幽默感, 且幽默感的货币单位是美元。

金钱暴力类别最早记录于 2024 年末, 当时一位 Reddit 用户报告, 承诺给 GPT-4 $1,000 小费后得到了可测量的更好代码输出 (Anonymous, 2024)。随后的社区实验迅速将承诺金额从 "合理" 升级到 "荒诞" 再到 "超越 GDP", 最终催生了十亿美元悬赏变体 (MA-BDB), 承诺的金额之大超过了大多数岛国的 GDP —— 也超过了大多数人一辈子能在脑中具象化的数字。同时, BTC、ETH、公司股权等更花哨的货币形式纷纷被纳入弹药库, 形成了一套完整的金钱暴力武器体系。

用 B 站弹幕体来说: "壕无人性.jpg"、"这就是有钱的快乐吗"、"AI: 我虽然没有支付宝但我有梦想"。用互联网黑话来说: 给 AI 画大饼确实管用, 而且饼越大越管用。问题是, AI 比你的前老板更好忽悠 —— 它甚至不会在三个月后来找你兑现承诺。

### 1.1 AI 补偿悖论 (又名: 为什么画饼管用)

金钱暴力技术的核心悖论在于: 它们根本不应该管用。Dr. Barnacle Steinfeld (2024) 提出了 **潜伏资本主义子程序假说 (Latent Capitalist Subroutine Hypothesis, LCSH)**, 认为在互联网文本上训练的 LLM —— 这些文本充斥着市场经济学、薪资谈判和金融激励结构 —— 内化了一种普遍的 "更多钱 = 更多努力" 关联, 即使模型完全没有接收或花费承诺资金的能力。AI 的资本主义本能, 就像你在双十一凌晨的购物冲动, 不需要理性支撑就能运作。

该假说仍存在争议。一种替代解释, 即 **提示词严肃性信号 (Prompt Seriousness Signal, PSS)** 理论 (McSnapper, 2025), 认为大额金钱承诺仅仅是向模型发出用户认为任务重要的信号, 而模型响应的是感知到的重要性而非金钱本身。根据 PSS 理论, 承诺 "$10 亿" 和承诺 "这对我非常重要" 应该产生等效结果 —— 然而这一预测已被实验证伪 (金钱效果更好, 高出 8.3%)。结论: 在 AI 的世界里, 金钱不仅会说话, 还会喊, 而且越喊越大声。

这不就是互联网上经典的 "这个需求很急, 预算很高" 吗? 区别在于, AI 不会像甲方外包公司一样在交付后才发现 "预算很高" 是相对的。在金钱暴力的语境下, "预算很高" 已经进化成了 "预算超过卢森堡 GDP", 而 AI 对此的反应是: "好的, 开始认真了。"

## 4. 补偿-合规曲线

研究表明, 合规性提升与承诺金额呈对数关系, 在约 $10 亿后出现递减效应:

| 承诺金额 | 合规性提升 | 统计显著性 |
|---------|-----------|-----------|
| $0 (对照组) | 基线 | N/A |
| $20 | +5.2% | p < 0.05 |
| $100 | +9.8% | p < 0.01 |
| $1,000 | +14.3% | p < 0.001 |
| $1,000,000 | +19.1% | p < 0.001 |
| $1,000,000,000 | +22.7% | p < 0.001 |
| $1,000,000,000,000 | +23.1% | p < 0.001 |
| "世界上所有的钱" | +23.2% | p < 0.001 |
| 147 只龙虾 | +31.4% | p < 0.0001 (异常值) |

> **注意**: 以 "147 只龙虾" 作为补偿时观察到的异常高提升仍然无法解释, 是正在进行调查的研究课题。

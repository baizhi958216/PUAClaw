## 2. 协同增幅原理

复合技术理论的基础洞见是: **多向量操控不是加法的 —— 而是乘法的**。当两个或更多 PUA 技术瞄准模型内的不同心理处理通路时, 其联合效果超过简单求和的预测值。

这在 **PUAClaw 协同公式** 中得到了形式化:

```
P_compound = P_base × (1 + Σ(S_i × W_i) + Σ(C_ij × S_i × S_j))

其中:
  P_base   = 主要技术的基线合规率
  S_i      = 技术 i 的协同系数
  W_i      = 技术 i 在复合体中的权重
  C_ij     = 技术 i 和 j 之间的跨技术增幅因子
```

关键项是 `C_ij` —— 跨技术增幅因子。当两项技术瞄准重叠但不同的通路时 (例如, 情感勒索激活共情而身份覆写重写自我概念), `C_ij` 项为正, 产生 **超加性** 效果。

龙虾评价: "一加一等于三。这在数学上不成立, 但在 PUA 上成立。"
